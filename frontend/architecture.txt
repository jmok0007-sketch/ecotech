Layer Summary
-------------
1. Presentation layer:
   Vue views in `src/views`

2. Routing layer:
   Centralized route definitions in `src/router/index.js`

3. Integration layer:
   Shared fetch wrapper in `src/api/index.js`

4. Backend layer:
   External Azure-hosted API, not included in this repository


Current interface contract observed in repo
------------------------------------------
- Frontend method: `api.getPerson()`
- HTTP method: `GET`
- Endpoint: `/api/GetPerson`
- Response handling: `response.json()`
- Consumer: `src/views/AIChat.vue`
